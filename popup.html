Accessibility Personas

Choose a persona to simulate the experience of a web user with accessibility-related needs.

Ashleigh

A severely sight-impaired person with a screen reader.

The current tab's content is now blurred to simulate Ashleigh's sight impairment.

Open a screen reader on your device to experience how Ashleigh navigates the page. On Mac, do this in the System Settings by activating VoiceOver.

Pawel

An autistic person who is easily distracted when browsing the web.

You'll see images and moving text, simulating Pawel's distractions.

Consider installing additional extensions:

1. Add sound distractions with Chrome Sound Effects . Use headphones to hear them loud enough.

2. Adjust colors in the browser with the Midnight Lizard extension, which Pawel uses to help him focus.

Learn more about Pawel

Simone

A dyslexic person who needs more time to understand text and forms.

The text in the current tab is being scrambled to illustrate Simone's experience with dyslexia.

Try to read paragraphs or navigate forms with this setting. Notice which parts of the page are especially hard to read.

Learn more about Simone

Ron

An older person with multiple conditions and no assistive tech.

The current tab is blurred and bright to simulate Ron's vision impairment. The cursor wiggles, making it harder to select items.

Note: If the original cursor is still visible, try moving it to the address bar and back in.

Check if the contrasts on the page are still sufficient and if all text can be read. Notice how much easier it is when the most important information is mentioned upfront.

Learn more about Ron

Claudia

A sight-impaired person who uses a screen magnifier.

Claudia needs to magnify the page content to see everything more clearly. The current tab is now zoomed in to 300%.

Adjust your system settings to improve the simulation. On Mac, navigate to "Accessibility > Display", set your cursor to maximum size and increase contrast to the maximum.

Learn more about Claudia

Chris

A person with rheumatoid arthritis who mostly uses his keyboard.

As Chris does not see colors well, your tab is shown in greyscale. Your mouse cursor is hidden because Chris only uses his keyboard.

Note: If the cursor is still visible, try moving it to the address bar and back in.

Put your mouse aside and only use the keyboard to navigate the page. Use the TAB and ENTER keys and pay attention to the order in which elements are selected.
